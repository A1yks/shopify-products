### Installing

To install all dependencies, run the following command:

```bash
npm run install-all
```

### Running the Application

To start the application in development mode, run:

```bash
npm run dev
```

To build the application, run:

```bash
npm run build
```

To start the application, run:

```bash
npm start
```
